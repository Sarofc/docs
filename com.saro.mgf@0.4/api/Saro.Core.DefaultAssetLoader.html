    Class DefaultAssetLoader

    不可重复资源加载器

    精细管理所有资源引用，保证 此加载器 每个资源只被加载一次。持有此加载器的对象生命周期结束时，调用释放接口 UnloadAllAssetRef() 
    一般在 `自动卸载AB` 的情况下使用
